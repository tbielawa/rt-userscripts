Fixed Action Menu for Request Tracker 3.6

Install a Userscript for Request Tracker 3.6 which duplicates the action menu as a fixed menu on the bottom of a browser window when you are viewing a ticket.. Install it!



Add Starting New Lines and Auto Select Comment Box

I made this to make commenting a little quicker. Request Tracker will start your signature at character 0 of the <textarea> for comments/replies, I didn't like that behavior. Instead this userscript adds two empty new lines at the beginning of the comment block and auto-selects it when the page has loaded. It also increases the default height and width of the textarea. Install it!



Trouble Installing?

If you're using Firefox you'll need a Userscript plugin like Greasemonkey for it to work.

Google Chrome handles Userscripts out of the box you just have to go through the irritating task of enabling them. Mac OS X Hints has a brief article on how to enable user scripts in Chrome. On OSX you'll have to make a ~/Library/Application Support/Google/Chrome/Default/User Scripts/ before it will work.
